The **Admin Dashboard** is a restricted interface designed for the **Platform Owner** (Self-Hoster). Unlike the User Dashboard (which manages agents), this dashboard manages the **Operating System** of the platform itself.

You access it at `/admin` (e.g., `https://app.your-domain.com/admin`).

## System Health

Upon logging in, the **App Overview** provides a heartbeat check of your cluster.

<img src="/images/platform-admin-overview.jpg" alt="Admin Dashboard Overview" className="mt-0 mb-6 rounded-xl w-full border shadow-sm" />

### Understanding the Metrics
*   **System Version:** The version of the running **Frontend** binary.
*   **Database Schema:** The version of the **MongoDB** structure.
    *   *Critical:* If these versions mismatch, the system will trigger a [Migration Lock](/developers/self-hosting/lifecycle).
*   **Migration Status:** Indicates if the background processor is currently modifying the database structure.
